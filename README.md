# Reel Interpreter

Understand French Instagram Reels and TikToks — slang, humor, and all.

A language-learning tool that helps non-native speakers deeply understand French short-form social media content, including translations, slang breakdowns, cultural context, and meme format explanations.

## Setup

```bash
bun install
cp .env.example .env.local
# Add your Gemini API key to .env.local
bun dev
```

## Tech Stack

- Next.js 16 (App Router)
- Google Gemini 2.0 Flash
- shadcn/ui + Tailwind CSS
- TypeScript
